## 📌 À propos / حول المشروع

**Tassaout System** est la plateforme de gestion numérique de l'agence immobilière **تساوت العقارية**, basée à Kelaat Sraghna, Maroc.

Le système est géré par **Alpha Core Nexus** avec l'assistance de **Dana**, une IA dédiée à l'automatisation des opérations immobilières.

> **نظام تساوت** هو المنصة الرقمية لإدارة العمليات العقارية في قلعة السراغنة.
> يدار بواسطة: **Alpha Core Nexus** | المستشارة: **Dana**

## 🖥️ Tableau de Bord Administrateur

Le fichier `pages/admin.html` est l'interface centrale du système Tassaout. Il comprend cinq sections :

| Section | Description |
|---------|-------------|
| 📊 **Statistiques Globales** | Vue synthétique : nombre de propriétés, clients actifs, transactions, revenus |
| 🏠 **Gestion des Propriétés** | Ajout, consultation et suivi des biens immobiliers (adresse, type, prix, statut) |
| 👥 **Gestion des Clients** | Répertoire clients avec coordonnées, statut et historique |
| 📈 **Analytique** | Rapports de performance et tendances du marché *(en cours)* |
| ⚙️ **Paramètres** | Configuration système *(en cours)* |

> L'interface est alimentée par `js/admin.js` et stylisée via `css/admin.css`.
